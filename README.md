# MonieCredit AI

## Project Overview

MonieCredit AI is an end-to-end Consumer Lending Decision Platform designed to simulate how modern fintech companies such as Moniepoint evaluate loan applications, assess customer risk, determine credit limits, detect fraud, and monitor loan portfolios using Data Science and Machine Learning.

This project is being built as a production-style portfolio to demonstrate real-world skills in SQL, Python, Machine Learning, Data Engineering, Power BI, API development, and MLOps.

---

# Business Problem

Financial institutions receive thousands of loan applications every day. Making lending decisions manually is slow, inconsistent, and increases the risk of loan defaults.

MonieCredit AI provides a data-driven solution that helps lenders:

- Evaluate customer creditworthiness
- Predict loan default risk
- Recommend suitable credit limits
- Detect fraudulent activities
- Monitor customer repayments
- Improve profitability while reducing credit losses

---

# Project Objectives

The primary objectives of this project are:

- Build a scalable consumer lending database
- Generate realistic synthetic financial data
- Develop machine learning models for credit risk prediction
- Build a credit limit recommendation engine
- Create fraud detection models
- Deploy prediction APIs using FastAPI
- Visualize lending performance using Power BI

# Installation

Clone the repository

```bash
git clone https://github.com/tunde-samson/MonieCredit-AI.git
```

Create a virtual environment

```bash
python -m venv .venv
```

Activate it

Windows

```bash
.venv\Scripts\activate
```

Install dependencies

```bash
pip install -r requirements.txt
```
